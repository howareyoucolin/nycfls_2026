# Hello World PWA

Minimal test PWA for quick phone testing.

## Run

```bash
cd adminApp
npm run start
```

Then open:

`http://localhost:18086`

If you want to test on your phone, use your Mac's LAN IP instead of `localhost`, for example:

`http://192.168.1.20:18086`

## What it includes

- installable web app manifest
- service worker
- standalone app shell
- simple Hello World UI
